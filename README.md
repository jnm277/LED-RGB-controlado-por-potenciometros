# LED-RGB-controlado-por-potenciometros
LED rgb que é controlado por 3 potenciômetros, contém circuito e código.
<img width="1518" height="760" alt="Image" src="https://github.com/user-attachments/assets/66344885-f2c3-40cd-8e2d-4d332e32c7cb" />
Conexões dos pinos:<br>
>Potenciômetro 1 => A0;<br>
>Potenciômetro 2 => A1;<br>
>Potenciômetro 3 => A2;<br>
>LED RGB => pinos digitais 11, 10 e 9.<br>

Valor das resistências:<br>
>Potenciômetros => 10k ohm;<br>
>Resistor => 10k ohm.<br>
